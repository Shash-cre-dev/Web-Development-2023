TO ADD COMMENTS IN HTML
CTRL + /

TO COPY THE SAME ELEMENTS DOWN THE ROW
CTRL + D

TO AUTO-COMPLETE ATTRIBUTES
TAB

TO ADD MULTIPLE ELEMENTS OF THE SAME TYPE IN A SINGLE GO IN HTML 
ELEMENT_TAG*X, where X is the number of elements required
-TO MOVE BETWEEN MULTIPLE ELEMENTS - TAB
(only be used when creating multiple elements at same time)

TO SELECT AN ITEM AND MOVE IT UP OR DOWN WITHOUT CUT/COPY
-Select the Item
-use ALT + DOWNWARD ARROW - For moving Down
-use ALT + UPWARD ARROW - For moving up
